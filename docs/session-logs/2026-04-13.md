# Session Log — 2026-04-13

## Sesión — Migración completa Supabase → DigitalOcean + Thumbnails Sketchfab

### Problema raíz
Supabase Auth tenía un bug crítico: `signInWithPassword` retornaba HTTP 200 del servidor pero el SDK JavaScript nunca resolvía la Promise en el browser. Múltiples intentos de fix fallaron (timeout wrappers, disable realtime, removeAllChannels, detectSessionInUrl). El usuario decidió migrar completamente fuera de Supabase.

### Solución implementada
Migración total a infraestructura propia en DigitalOcean:
- **Droplet**: Ubuntu 24.04, IP 159.203.189.167
- **Backend**: Express + PostgreSQL 16 + JWT auth (bcryptjs + jsonwebtoken)
- **Storage**: DigitalOcean Spaces (galeria-3d-files, nyc3)
- **Proxy**: Nginx sirve frontend estático, proxy /api/ a Express, proxy /cdn/ a DO Spaces CDN
